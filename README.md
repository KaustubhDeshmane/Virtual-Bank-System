# 🏦 Virtual Bank System

A desktop banking application built using **Java Swing**, **JDBC**, and **MySQL**. The system enables users to manage accounts, perform transactions, transfer funds, view transaction history, and update profile information through an interactive graphical interface.

---

## ✨ Features

### 👤 User Management

* New Customer Registration
* Customer Login
* Admin Login
* Profile Management

### 💰 Banking Operations

* Deposit Money
* Withdraw Money
* Fund Transfer
* Balance Inquiry
* Passbook / Transaction History

### 📊 Transaction Tracking

* Automatic Transaction Recording
* Transaction History with Date & Time
* Real-Time Balance Updates

* Java

## 🚀 Installation

### Prerequisites

* Java JDK 8+
* MySQL Server
* IntelliJ IDEA / Eclipse

### Setup

1. Clone the repository

```bash
git clone https://github.com/KaustubhDeshmane/Virtual-Bank-System.git
```

2. Create the database using:

```text
database/bank.sql
```

3. Update MySQL credentials in the source code.

4. Run:

```text
Landing.java
```
