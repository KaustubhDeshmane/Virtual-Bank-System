# Virtual Bank System 🏦

A comprehensive desktop banking application built with **Java Swing** and **MySQL** that simulates real banking operations with multiple user roles and financial services.

## ✨ Features

### 👤 Multi-User System
- **Customer Portal** - Account management and transactions
- **Bank Employee Dashboard** - Customer service operations
- **Admin Panel** - Full system control and oversight

### 💰 Banking Operations
- **Account Management** - Create, view, and manage bank accounts
- **Fund Transfers** - Secure inter-account transfers
- **Transaction History** - Detailed record of all financial activities
- **Balance Inquiry** - Real-time account balance checking
- **Loan Processing** - Loan application and management system
- **Fixed Deposits** - Term deposit management

### 🔒 Security Features
- **Secure Login** - Multi-factor authentication
- **Transaction PIN** - Additional security layer
- **Activity Logging** - Comprehensive audit trails
- **Encrypted Data** - Secure data storage

### 📊 Financial Services
- **Interest Calculation** - Automatic interest on savings
- **Account Statements** - Generate and view statements
- **Cheque Book Requests** - Digital cheque management
- **Online Services** - Bill payments and utilities

## 🚀 Quick Start

### Prerequisites
- Java JDK 8 or higher
- MySQL Server 5.7+
- MySQL Connector/J driver

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/asheesh109/Virtual-Bank-System.git
   ```

2. **Database Setup**
   - Run the SQL scripts to create the database schema
   - Configure database connection settings

3. **Run Application**
   - Import project into your Java IDE
   - Add MySQL Connector/J to libraries
   - Run the main class

### Default Access
- **Admin**: Full system control
- **Employee**: Customer management access
- **Customer**: Personal banking operations

## 🎯 Usage Guide
1. **Login** with appropriate credentials
2. **Navigate** through the user-specific dashboard
3. **Perform banking operations** as per your role
4. **View reports** and transaction history
5. **Logout** securely after use

## 🛠️ Technology Stack
- **Java Swing** - Desktop GUI framework
- **MySQL** - Relational database management
- **JDBC** - Database connectivity layer
- **AWT** - Additional GUI components


## 🔧 Configuration
Update the `database.properties` file with your MySQL credentials:
```properties
db.url=jdbc:mysql://localhost:3306/bank_db
db.username=your_username
db.password=your_password
```
